# Modularization

该工程整理和收集了一些关于组件化的资料，主要包括一些组件化相关的介绍文章，以及Module配置，Gradle配置的知识，以及路由框架的一些开源框架及实现的原理，希望对大家学习理解组件化能够有一些帮助。
